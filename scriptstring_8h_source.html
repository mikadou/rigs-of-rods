2 // CScriptString
3 //
4 // This class is used to pass strings between the application and the script engine.
5 // It is basically a container for the normal std::string, with the addition of a
6 // reference counter so that the script can use object handles to hold the type.
7 //
8 // Because the class is reference counted it cannot be stored locally in the
9 // application functions, nor be received or returned by value. Instead it should
10 // be manipulated through pointers or references.
11 //
12 // Note, because the internal buffer is placed at the beginning of the class
13 // structure it is infact possible to receive this type as a reference or pointer
14 // to a normal std::string where the reference counter doesn't have to be manipulated.
15 //
