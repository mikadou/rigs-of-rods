Detailed Description

Processes rig-file-parser output into actual simulation data structures.

Function writing convention:

  • Process*(Definition & def) Transform elements of .truck file to rig structures.
  • FindAndProcess*(Definition & def) Find and process an element which should be unique in the current vehicle configuration.
  • Add*(), Create*() or Build*() Add partial structures to rig.
  • Other functions are utilities.
Author
Petr Ohlidal
