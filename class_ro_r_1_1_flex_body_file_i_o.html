Detailed Description

Enables saving and loading flexbodies from/to binary file.

FILE STRUCTURE: 1. Signature 2. Metadata

See also:
FlexBodyFileMetadata 3. Flexbodies a. Header
FlexBodyRecordHeader b. Data (not present if flexbody has flags IS_FAULTY==true or IS_ENABLED==false) 1. Locator list 2. Positions buffer 3. Normals buffer 4. Colors buffer (only present if flag HAS_TEXTURE_BLEND == true)
