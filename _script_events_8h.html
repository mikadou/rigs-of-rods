enum scriptEvents

This enum describes what events are existing. The script can register to receive events.

Enumerator:
SE_COLLISION_BOX_ENTER 

triggered when truck or person enters a previous registered collision box, the argument refers to the collision box ID

SE_COLLISION_BOX_LEAVE 

triggered when truck or person leaves a previous registered and entered collision box, the argument refers to the collision box ID

SE_TRUCK_ENTER 

triggered when switching from person mode to truck mode, the argument refers to the truck number

SE_TRUCK_EXIT 

triggered when switching from truck mode to person mode, the argument refers to the truck number

SE_TRUCK_ENGINE_DIED 

triggered when the trucks engine dies (from underrev, water, etc), the argument refers to the truck number

SE_TRUCK_ENGINE_FIRE 

triggered when the planes engines start to get on fire, the argument refers to the truck number

SE_TRUCK_TOUCHED_WATER 

triggered when any part of the truck touches water, the argument refers to the truck number

SE_TRUCK_BEAM_BROKE 

triggered when a beam breaks, the argument refers to the truck number

SE_TRUCK_LOCKED 

triggered when the truck got lock to another truck, the argument refers to the truck number

SE_TRUCK_UNLOCKED 

triggered when the truck unlocks again, the argument refers to the truck number

SE_TRUCK_LIGHT_TOGGLE 

triggered when the main light is toggled, the argument refers to the truck number

SE_TRUCK_SKELETON_TOGGLE 

triggered when the user enters or exits skeleton mode, the argument refers to the truck number

SE_TRUCK_TIE_TOGGLE 

triggered when the user toggles ties, the argument refers to the truck number

SE_TRUCK_PARKINGBREAK_TOGGLE 

triggered when the user toggles the parking break, the argument refers to the truck number

SE_TRUCK_BEACONS_TOGGLE 

triggered when the user toggles beacons, the argument refers to the truck number

SE_TRUCK_CPARTICLES_TOGGLE 

triggered when the user toggles custom particles, the argument refers to the truck number

SE_TRUCK_GROUND_CONTACT_CHANGED 

triggered when the trucks ground contact changed (no contact, different ground models, etc), the argument refers to the truck number

SE_GENERIC_NEW_TRUCK 

triggered when the user spawns a new truck, the argument refers to the truck number

SE_GENERIC_DELETED_TRUCK 

triggered when the user deletes a truck, the argument refers to the truck number

SE_GENERIC_INPUT_EVENT 

triggered when an input event bound to the scripting engine is toggled, the argument refers to event id

SE_GENERIC_MOUSE_BEAM_INTERACTION 

triggered when the user uses the mouse to interact with the truck, the argument refers to the truck number

SE_TRUCK_TRACTIONCONTROL_TOGGLE 

triggered when the user toggles the tractioncontrol system, the argument refers to the truck number

SE_TRUCK_ANTILOCKBRAKE_TOGGLE 

triggered when the user toggles the antilockbrake, the argument refers to the truck number

SE_ANGELSCRIPT_MANIPULATIONS 

triggered when the user tries to dynamically use the scripting capabilities (prevent cheating)
